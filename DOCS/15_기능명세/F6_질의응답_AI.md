# F6 질의응답 AI

## 범위
- `POST /qa`
- LangChain splitter 기반 근거 선택
- Gemini 응답 + fallback 응답

## 구현 원칙
- 질문 입력은 500자 이하로 제한합니다.
- 입력은 `question`, `context` 구조를 사용합니다.
- 답변은 `[답변]`, `[근거 요약]`, `[판단]` 섹션으로 정규화합니다.
- 근거 snippet은 `rag_top_k` 범위로 반환합니다.
- 모델 응답이 비정형이더라도 형식을 정규화합니다.
- 문서/학습 도메인 밖 질문은 차단 응답으로 처리합니다.
